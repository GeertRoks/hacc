## sgutil run

<code>
  sgutil run [arguments [flags]] [--help]
</code>
<p>
  &nbsp; &nbsp; Executes the accelerated application on a given device.
</p>

### Arguments

* [vitis](./sgutil-run-vitis.md)
* [vivado]()

<code>
  -h, --help
</code>
<p>
  &nbsp; &nbsp; Help to use this command.
</p>
