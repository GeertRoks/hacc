## sgutil get

<code>
  sgutil get [arguments [flags]] [--help]
</code>
<p>
  &nbsp; &nbsp; Retreives information from the server/s.
</p>

### Arguments

* [device](./sgutil-get-device.md)
* [ip](./sgutil-get-ip.md)
* [mac](./sgutil-get-mac.md)
* [serial](./sgutil-get-serial.md)

<code>
  -h, --help
</code>
<p>
  &nbsp; &nbsp; Help to use this command.
</p>

### Examples
```
$ sgutil get -h
$ sgutil get ip
$ sgutil get serial
```
