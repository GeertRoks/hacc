## sgutil new

<code>
  sgutil new [arguments] [--help]
</code>
<p>
  &nbsp; &nbsp; Creates a new project of your choice.
</p>

### Arguments

* [vitis](./sgutil-new-vitis.md)
* [vivado](./sgutil-new-vivado.md)

<code>
  -h, --help
</code>
<p>
  &nbsp; &nbsp; Help to use this command.
</p>

### Examples
```
$ sgutil new -h
$ sgutil new vitis
$ sgutil new vivado
```
